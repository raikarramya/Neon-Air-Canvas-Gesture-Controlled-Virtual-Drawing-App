# ✨ Neon Air Canvas – Gesture Controlled Virtual Drawing App

A futuristic **AI-powered virtual drawing application** that allows users to draw in the air using **hand gestures** and real-time **computer vision tracking**.

Built using **MediaPipe Hands**, **JavaScript**, **HTML5 Canvas**, and **Webcam Hand Tracking** technology.

---

# 🚀 Project Overview

Neon Air Canvas transforms your webcam into a gesture-based drawing interface.

Using hand landmark detection, users can:

- ✍️ Draw in the air with finger gestures
- 🖐️ Erase drawings using palm gestures
- 🎨 Change neon colors
- 📏 Adjust brush size dynamically
- 👋 Control the canvas without touching the screen

The project provides a smooth futuristic UI with glowing neon effects and real-time hand tracking.
